Sequences: 
	What: Incrementing integers
		- 1, 2, 3, 4, etc...
		Can specify 
			- start value 
		
	Why: 
		Autogenerated unique numbers for surrogate keys. 
			- surrogate key:
				- Meaningless value, usually generated by the system.
		i.e. PET_ID, _PET_RES_ID
		
	How: 
		CREATE SEQUENCE sequence_name
			MINVALUE 		value
			MAXVALUE 		value
			START WITH		value
			INCREMENT BY	value
			CACHE			value
			
		example: 
		CREATE SEQUENCE supplier_seq
			MINVALUE 1
			START WITH 1
			INCREMENT BY 1
			
		INSERT INTO supplier
			(supplier_id, supplier_name)
		VALUES
			(supplier_seq.NEXTVAL, 'Kraft Foods');
			
		INSERT INTO products
			(product_id, supplier_id, product_name)
		VALUES
			(12, supplier_seq.CURRVAL, 'Mac N Cheese');
